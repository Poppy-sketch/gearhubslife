## Customization

### To Modify Colors:
Edit the CSS `:root` variables at the top of each page:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #60a5fa;
    /* ... */
}
```

### To Add New Pages:
1. Copy an existing page as template
2. Update the `<title>` and `<h1>` tags
3. Modify the content
4. Add links to navigation menu
5. Maintain consistent styling
